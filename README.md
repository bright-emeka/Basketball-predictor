# Basketball Predictor

This repository has been converted into a React application using `create-react-app` style structure.

## Available Scripts

In the project directory, you can run:

### `npm install`

Installs the app dependencies.

### `npm start`

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Features

- Enter two NBA team names.
- Enter your API-Sports API key.
- Predict the highest scoring quarter using recent games and optional head-to-head data.
- Predict the lowest scoring quarter using head-to-head games when available, otherwise recent games.

## Notes

- The app uses the Basketball API from [API-Sports](https://api-sports.io/).
- If the API blocks browser requests due to CORS, a backend proxy may be required.
